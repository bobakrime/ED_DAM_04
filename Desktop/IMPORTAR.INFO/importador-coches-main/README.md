# ImportCalc - Calculadora de Importación de Coches

Esta es una aplicación web construida con Next.js y Tailwind CSS para calcular el coste de importar coches de Alemania a España.

## Características

- **Calculadora de Costes**: Estima el precio final incluyendo IVA, impuesto de matriculación, transporte y gestoría.
- **Diseño Premium**: Interfaz moderna y responsive con modo oscuro.
- **Páginas**:
  - Inicio: Búsqueda y explicación del proceso.
  - Resultados: Desglose detallado de costes.
  - Contacto: Formulario para solicitar ayuda profesional.

## Tecnologías

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)

## Cómo Ejecutar

1.  Instalar dependencias:
    ```bash
    npm install
    ```

2.  Ejecutar el servidor de desarrollo:
    ```bash
    npm run dev
    ```

3.  Abrir [http://localhost:3000](http://localhost:3000) en tu navegador.

## Estructura del Proyecto

- `app/page.tsx`: Página de inicio.
- `app/results/page.tsx`: Página de resultados (mockup con datos de ejemplo).
- `app/contact/page.tsx`: Página de contacto.
- `app/globals.css`: Configuración de estilos globales y tema de Tailwind.
